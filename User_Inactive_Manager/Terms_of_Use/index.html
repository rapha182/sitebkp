Inactive Users Manager – Terms of Use

Usage conditions, responsibilities, limitations, and contact details for Inactive Users Manager.

Acceptance of Terms

By installing or using this app, you agree to these terms.

Description

Inactive Users Manager identifies users without recent activity and allows administrators to export reports, review inactivity, and remove users from groups when permitted.

Responsibilities

You are responsible for validating inactivity before taking action, ensuring compliance with internal policies, and managing user lifecycle correctly.

Limitations

The app cannot override Atlassian Guard / SCIM, depends on Atlassian API availability, and may rely on activity signals provided by Atlassian.

Availability

The app is provided “as is”, without guaranteed uptime.

Liability

The developer is not responsible for incorrect removals, misinterpretation of inactivity, or operational impacts.

Changes

Terms may be updated at any time.

Contact
